# A11y Demo

This project demonstrates key accessibility principles using Bootstrap CSS framework. Each page showcases different accessibility features and best practices.

## Pages and Features

1. **Home Page**: Overview and navigation
2. **Forms**: Accessible form controls and validation
3. **Components**: Accessible Bootstrap components
4. **Tables**: Accessible data tables
5. **Media**: Accessible images and media content

## Setup

1. Clone this repository
2. Open `index.html` in your browser
3. Navigate through different pages to learn about accessibility features

## Key A11y Features Demonstrated

- ARIA labels and roles
- Semantic HTML
- Keyboard navigation
- Color contrast
- Form accessibility
- Screen reader compatibility
- Responsive design
