WCAG 1.2.3 Audio Description or Media Alternative (Level A)

This page demonstrates how to provide audio descriptions or media alternatives for prerecorded video content.

Poor Implementation Example

Video Without Audio Description

Why this fails:
  • No audio description for important visual information
  • No text alternative available
  • Visual-only content is not accessible to blind users
  • Important context is missing

Good Implementation Example

Video With Audio Description and Text Alternative

Full Text Alternative

00:00 [Opening with colorful animation] Audio description helps blind and low vision audiences enjoy visual media.
00:15 [Person using screen reader] It provides descriptions of important visual information during natural pauses in dialogue.
00:30 [Examples of visual scenes being described] The narrator describes actions, characters, costumes, and other visual elements.
Why this works:
  • Audio description track available
  • Full text transcript with timestamps
  • Description toggle control
  • Accessible media controls
  • Visual descriptions included in transcript

Alternative Format Examples

Descriptive Transcript

A descriptive transcript includes both the audio content and descriptions of important visual information:

Scene 1: Introduction

[Animated introduction with text and graphics]
The video begins with an explanation of audio description and its importance for accessibility.

Scene 2: Demonstration

[Split screen showing content with and without audio description]
A comparison demonstrates how audio description enhances the viewing experience for blind and low vision audiences.

Scene 3: Implementation

[Examples of audio description in various media]
Various examples show how audio description is implemented in different types of visual content.

Audio Description Script

An audio description script shows exactly what will be spoken during natural pauses in the main audio:

00:05 - "Animated graphics appear showing accessibility symbols"

00:15 - "Person navigates content using assistive technology"

00:30 - "Split screen demonstrates audio description impact"

Best Practices Summary

  • Provide audio descriptions for important visual information
  • Include a complete text transcript
  • Make descriptions available but toggleable
  • Ensure media controls are keyboard accessible
  • Include timestamps for synchronization
  • Describe all meaningful visual content
