# 1.4.1 Use of Color (Level A)


## Bad Implementation

The form uses color (red text) as the sole indicator of required fields. Users who cannot perceive color differences 
might not realize which fields are required.

```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Bad Implementation</title>
    <style>
        .required {
            color: red;
        }
    </style>
</head>
<body>
    <form>
        <label for="name" class="required">Name:</label>
        <input type="text" id="name" name="name"><br>

        <label for="email" class="required">Email:</label>
        <input type="email" id="email" name="email"><br>

        <label for="phone">Phone:</label>
        <input type="tel" id="phone" name="phone"><br>

        <button type="submit">Submit</button>
    </form>
    <p>Fields in red are required.</p>
</body>
</html>

```
## Good Implementation

In addition to using color, an asterisk (*) is added next to the labels of required fields, and the 
aria-required="true" attribute is used to programmatically indicate that the field is required. 
The explanation below the form clarifies the meaning of the asterisk. 
This approach ensures that information about required fields is accessible to all users, regardless of their 
ability to perceive color.

```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Good Implementation</title>
    <style>
        .required {
            color: red;
        }
    </style>
</head>
<body>
    <form>
        <label for="name">Name: <span class="required">*</span></label>
        <input type="text" id="name" name="name" aria-required="true"><br>

        <label for="email">Email: <span class="required">*</span></label>
        <input type="email" id="email" name="email" aria-required="true"><br>

        <label for="phone">Phone:</label>
        <input type="tel" id="phone" name="phone"><br>

        <button type="submit">Submit</button>
    </form>
    <p><span class="required">*</span> indicates required fields.</p>
</body>
</html>

```
