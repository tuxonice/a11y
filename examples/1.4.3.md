# 1.4.3 Contrast (Minimum) (Level AA)

## Bad Implementation

The text color `#b0b0b0` against the background color #f0f0f0 has a low contrast ratio, making it difficult for users 
with visual impairments to read the text.

```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Bad Implementation</title>
    <style>
        body {
            background-color: #f0f0f0;
            color: #b0b0b0;
        }
    </style>
</head>
<body>
    <h1>Welcome to Our Website</h1>
    <p>Our mission is to provide quality services to all our customers.</p>
</body>
</html>
```

## Good Implementation

The text color is changed to `#333333`, which provides a higher contrast ratio against the background color `#f0f0f0`. 
This ensures better readability and accessibility for users with visual impairments.

```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Good Implementation</title>
    <style>
        body {
            background-color: #f0f0f0;
            color: #333333;
        }
    </style>
</head>
<body>
    <h1>Welcome to Our Website</h1>
    <p>Our mission is to provide quality services to all our customers.</p>
</body>
</html>
```
