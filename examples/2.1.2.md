# 2.1.2 "No Keyboard Trap" (Level A)

## Bad Implementation

The modal dialog opens and focuses on the close button, but there is no way to close it using the keyboard (other than clicking the button) or to move the focus back to the main content.

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Bad Implementation</title>
    <style>
        #modal {
            display: none;
            position: fixed;
            top: 50%;
            left: 50%;
            transform: translate(-50%, -50%);
            padding: 20px;
            background-color: white;
            border: 1px solid #ccc;
        }
    </style>
    <script>
        function openModal() {
            document.getElementById('modal').style.display = 'block';
            document.getElementById('closeButton').focus();
        }

        function closeModal() {
            document.getElementById('modal').style.display = 'none';
        }
    </script>
</head>
<body>
    <button onclick="openModal()">Open Modal</button>

    <div id="modal" role="dialog">
        <p>This is a modal dialog.</p>
        <button id="closeButton" onclick="closeModal()">Close</button>
    </div>
</body>
</html>
```

## Good Implementation

The modal can be closed using the "Escape" key, allowing users to exit the modal without using a mouse. The tabindex="-1" ensures the modal can be focused programmatically, and focus is returned to the "Open Modal" button when the modal is closed, preventing a keyboard trap.

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Good Implementation</title>
    <style>
        #modal {
            display: none;
            position: fixed;
            top: 50%;
            left: 50%;
            transform: translate(-50%, -50%);
            padding: 20px;
            background-color: white;
            border: 1px solid #ccc;
        }
    </style>
    <script>
        function openModal() {
            document.getElementById('modal').style.display = 'block';
            document.getElementById('closeButton').focus();
        }

        function closeModal() {
            document.getElementById('modal').style.display = 'none';
            document.getElementById('openModalButton').focus();
        }

        // Ensure ESC key closes the modal
        document.addEventListener('keydown', function(event) {
            if (event.key === 'Escape') {
                closeModal();
            }
        });
    </script>
</head>
<body>
    <button id="openModalButton" onclick="openModal()">Open Modal</button>

    <div id="modal" role="dialog" aria-labelledby="dialogTitle" tabindex="-1">
        <h2 id="dialogTitle">Modal Dialog</h2>
        <p>This is a modal dialog.</p>
        <button id="closeButton" onclick="closeModal()">Close</button>
    </div>
</body>
</html>

```
