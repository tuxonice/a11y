
# 1.1.1 - Non-text Content (Level A)

## Bad Implementation

The image lacks an alt attribute, which means screen readers will not be able to provide any 
information about the image to users who are blind or visually impaired.

```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Bad Implementation</title>
</head>
<body>
    <h1>Our Company Logo</h1>
    <img src="logo.png">
</body>
</html>
```

## Good Implementation

The alt attribute is added with a descriptive text, "Company logo featuring a stylized blue and green globe." 
This text provides a meaningful description of the image, 
allowing users who cannot see the image to understand its content and purpose.

```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Good Implementation</title>
</head>
<body>
    <h1>Our Company Logo</h1>
    <img src="logo.png" alt="Company logo featuring a stylized blue and green globe">
</body>
</html>

```
